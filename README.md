# Poly Dash

Dash through vibrant street, dodge obstacles in this fast-paced 3D endless runner. Unlock new characters and show off your agility. Play now!

## Features

- **Endless Running Mechanics**: The game progressively gets faster and harder the longer you survive.
- **3D Perspective Track**: Built entirely using CSS 3D transforms (`preserve-3d` and `rotateX`), creating a compelling receding horizon effect.
- **Realistic Hurdles**: 3D traffic barricades to dodge, built purely with HTML/CSS.
- **Character Selection**: Choose between 4 distinct characters via a sleek selection carousel menu.
- **Synthesized 8-Bit Audio**: Real-time sound generation for moving, crashing, and scoring using the Web Audio API without needing external sound files!
- **Keyboard & Touch Controls**: Play on desktop using Arrow Keys / A & D or Spacebar (pause), or on mobile via left/right screen taps.

## Tech Stack

- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Lucide Icons**

## Getting Started

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the local development server
4. Open the provided localhost link in your browser

Have fun dashing!
